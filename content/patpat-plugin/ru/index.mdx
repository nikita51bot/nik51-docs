## Введение

Плагин PatPat представляет собой реализацию серверной части модификации PatPat,
который используется на серверах с ядрами Bukkit, Spigot, Paper, Folia и их форков.

<Callout type="warn">
Плагин **НЕ** заменяет собой функционал оригинального мода!

Это значит, что если игроки заходят на сервер без установленной модификации, то они не увидят
поглаживания других игроков, внезависимости, установлен ли плагин на сервере.
</Callout>

## Установка плагина

Плагин устанавливается аналогичным образом, как и другие плагины.
Для этого необходимо скачанный файл поместить в папку plugins на сервере.
В конце должна получится следующая структура

import { File, Folder, Files } from 'fumadocs-ui/components/files';

<Files>
  <Folder name="server-folder" defaultOpen>
    <Folder name="plugins" defaultOpen>
      <File className="text-fd-primary" name="patpat-plugin-1.2.5.jar" />
    </Folder>
    <Folder className="text-fd-muted-foreground" name="world" disabled />
    <Folder className="text-fd-muted-foreground" name="world_nether" disabled />
    <Folder className="text-fd-muted-foreground" name="world_the_end" disabled />
    <File name="server.jar" />
    <File name="server.properties" />
    <File name="start.sh" />
    <File name="global.css" />
  </Folder>
</Files>

После этого плагин будет работать!
