Список предназначен для ограничения или наоборот, предоставления права поглаживания
игроков путем добавления игроков (или их uuid) в данный список и выбора его режима
работы. Данный список хранится в файле `player-list.txt` в папке конфигов и представляет
из себя список игроков в формате:

```markdown title="player-list.txt"
a4369612-4870-3833-9a0b-1f1073297c22 nikita51
192e3748-12d5-4573-a8a5-479cd394a1dc LopyMine
...
```

### Команды управления

#### Просмотр состава списка [#info]

```shell
/patpat list info
```

Показывает текущих игроков в списке и активный режим работы.

#### Добавление игрока [#add]

```bash
/patpat list add "player|uuid"
```

Добавляет игрока в список по имени или uuid.

#### Удаление игрока [#remove]

```shell
/patpat list remove "player|uuid"
```

Удаляет игрока из списка.

#### Режимы работы [#set]

```shell
/patpat list set "mode"
```

Изменяет режим работы на один из трёх доступных:

- `BLACKLIST` — только игроки вне списка могут поглаживать
- `WHITELIST` — только игроки из списка могут поглаживать
- `DISABLED` — любой игрок может поглаживать
