## Introduction

The PatPat plugin is a server-side implementation of the PatPat modification,
designed for servers running Bukkit, Spigot, Paper, Folia, and their forks.

<Callout type="warn">
The plugin does **NOT** replace the functionality of the original mod!

This means that if players join the server without the mod installed, they will not see
other players being patted, regardless of whether the plugin is installed on the server.
</Callout>

## Plugin Installation

The plugin is installed in the same way as other plugins.
Place the downloaded file into the server's `plugins` folder.
Your final structure should look like this:

import { File, Folder, Files } from 'fumadocs-ui/components/files';

<Files>
  <Folder name="server-folder" defaultOpen>
    <Folder name="plugins" defaultOpen>
      <File className="text-fd-primary" name="patpat-plugin-1.2.5.jar" />
    </Folder>
    <Folder className="text-fd-muted-foreground" name="world" disabled />
    <Folder className="text-fd-muted-foreground" name="world_nether" disabled />
    <Folder className="text-fd-muted-foreground" name="world_the_end" disabled />
    <File name="server.jar" />
    <File name="server.properties" />
    <File name="start.sh" />
  </Folder>
</Files>

After that, the plugin will be ready to use!
